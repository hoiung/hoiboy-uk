{{- /* Pick the hero image for a post bundle. Priority order: 1. hero.* (explicit author override) 2. 2e1ax_vintage_entry_* (AdventureAnd.Me Joomla "main entry" prefix) 3. *main* (general convention) 4. first raster image resource (jpg/jpeg/png/webp/gif), alphabetical SVGs are EXCLUDED from candidate 4 because legacy bundles often contain inline diagram SVGs that are never standalone hero photos. Authors who want an SVG hero must rename it to hero.svg explicitly. Returns nil if the bundle has no eligible images. Callers must guard with `with`. */ -}} {{- $hero := .Resources.GetMatch "hero.*" -}} {{- if not $hero }}{{ $hero = .Resources.GetMatch "2e1ax_vintage_entry_*" }}{{ end -}} {{- if not $hero }}{{ $hero = .Resources.GetMatch "*main*" }}{{ end -}} {{- if not $hero -}} {{- range .Resources.ByType "image" -}} {{- if not $hero -}} {{- $ext := path.Ext .Name | lower -}} {{- if ne $ext ".svg" -}}{{ $hero = . }}{{- end -}} {{- end -}} {{- end -}} {{- end -}} {{- return $hero -}}
